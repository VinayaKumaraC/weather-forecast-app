# Weather Forecast Application

This project is a responsive weather forecast web application built using
HTML, Tailwind CSS, and JavaScript.

## Features
- Search weather by city name
- Fetch weather using current location
- Recent searched cities dropdown
- Temperature toggle (°C / °F)
- Extreme heat alert
- Dynamic background based on weather
- 5-day weather forecast
- Responsive design for mobile, tablet, and desktop

## Technologies Used
- HTML
- Tailwind CSS
- JavaScript
- OpenWeatherMap API

## How to Run
1. Open the project folder in VS Code
2. Replace `YOUR_API_KEY_HERE` in `script.js`
3. Open `index.html` in browser

## Learning Outcome
This project helped me understand API integration,
DOM manipulation, responsive UI design, and error handling.
